## Deskripsi Singkat Sistem Manajemen Kategori Buku
Sistem manajemen kategori buku berbasis PHP Native dan MySQL. Serta menggunakan UI simple dengan Boostrap.
Fitur:
1. Menampilkan data kategori
2. Menambah kategori
3. Mengedit Kategori
4. Menghapus Kategori
## Cara Menjalankan
1. Install XAMPP
2. Jalankan Apache dan MySQL 
3. Copy project ke folder instalasi XAMPP:
   `Biasanya di C:\xampp\htdocs\`
4. Import database
   `Buka phpMyAdmin dibrowser (localhost/phpmyadmin)`
   `Import file database_backup.sql`
5. Jalankan dibrowser
   `http://localhost/nama-folder-anda`
